# rsg-boostfps
Version for RedM / The gusti-boostfps script is to make players boost their fps in the game, to prevent fps drops, stuttering, etc. from occurring.

# Feature
- Boost Fps
- Graphics
- Others Mode
- Reset

# Dependencies
- ox_lib
- rsg-Core
  
Commands = 'boostfps' -- Commands open menu boostfps
